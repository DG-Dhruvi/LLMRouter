# Installation

## Requirements
- Python 3.10+
- Optional: CUDA GPU for faster training

## Install from PyPI
```bash
pip install llmrouter
```

## Install from source (editable)
```bash
git clone https://github.com/ulab-uiuc/LLMRouter.git
cd LLMRouter

conda create -n llmrouter python=3.10
conda activate llmrouter

pip install -e .
```

## Verify
```bash
llmrouter --version
```
